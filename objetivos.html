Objetivo general

Desarrollar una aplicación didáctica para el aprendizaje y evaluación de competencias de lengua castellana en el tema de Mitología griega,
para estudiantes de grado décimo de la institución educativa Francisco Antonio Zea.

Obgetivos especificos

  • Definir las necesidades del cliente para buscar una solución que satisfaga dicha necesidad.
  • Analizar las necesidades del cliente que permitan cumplir con los requerimientos del sistema.
  • Diseñar prototipos útiles para verificar y confirmar que cuenta con las características específicas planteadas.
  • Codificar el diseño a partir del lenguaje de programación elegido.
  • Probar la eficiencia del software identificando errores.
  • Validar la funcionalidad de la aplicación didáctica.
  • Evolucionar la aplicación didáctica para un mejor funcionamiento.
